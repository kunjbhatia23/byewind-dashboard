# ByeWind Dashboard

This is a modern and responsive dashboard application built with React, Vite, and Tailwind CSS. It features a clean and intuitive user interface with various data visualization components.

## ✨ Features

* **Responsive Design:** The dashboard is fully responsive and works on all screen sizes.
* **Data Visualization:** Includes line charts, bar charts, and donut charts to visualize data.
* **Component-Based:** Built with reusable React components.
* **Dark Mode:** Easily customizable for a dark mode theme.
* **Vite-Powered:** Fast development and build times with Vite.

## 🚀 Getting Started

### Prerequisites

* Node.js (v14 or higher)
* npm or yarn

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/kunjbhatia23/byewind-dashboard.git
    ```
2. Navigate to the project directory:
    ```bash
    cd byewind-dashboard
    ```
3. Install the dependencies:
    ```bash
    npm install
    ```

### Running the Project

To start the development server, run the following command:

```bash
npm run dev
```

The application will be available at [http://localhost:5173](http://localhost:5173).

## 🛠️ Technologies Used

- **React:** A JavaScript library for building user interfaces.
- **Vite:** A fast build tool for modern web projects.
- **Tailwind CSS:** A utility-first CSS framework for rapid UI development.
- **Chart.js:** A flexible charting library for data visualization.
- **Heroicons:** A set of free, high-quality SVG icons.

## 🎨 Customization

You can customize the application by modifying the following files:

- **src/index.css:** Add your own custom CSS styles.
- **src/components/**: Modify the existing components or create new ones to suit your needs.
